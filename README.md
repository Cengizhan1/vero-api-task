# VERO API TASK


## Requirements
```
- PHP 7.4 or higher
- SQLite database
```

## Installation
1. Clone the project:

```
git clone https://github.com/Cengizhan1/vero-api-task.git
``` 
2. Navigate to the project directory:

```
cd vero-api-task
``` 

    The application will run at http://localhost/vero-api-task/index.php by default.
    
    You are now ready to use the API.
